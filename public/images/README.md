# Image Assets

## Folder Structure

```
images/
├── hero/           # Homepage hero images
├── services/       # Service-related images
├── testimonials/   # Customer photos (with permission)
├── projects/       # Before/after project photos
└── team/           # Team member photos
```

## Image Guidelines

- **Format**: JPG for photos, PNG for logos/graphics
- **Optimization**: Compress images before uploading
- **Size**: Max 1920px wide for hero images, 800px for thumbnails
- **Naming**: Use descriptive names (e.g., `junk-removal-truck.jpg`, `before-garage-cleanout.jpg`)
